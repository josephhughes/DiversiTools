On this page, you will find answers to frequently asked questions. If you have any additional questions contact us.

I am getting an error message: "Error: The coding sequence for X is not a multiple of 3."

This is because the file you are using to assign the coding regions (-orf argument) is incorrectly formatted. You need to check the start and end positions of the protein X and make sure that the coding sequence is a multiple of three.
